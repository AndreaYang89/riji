# 腾讯云部署提示词

下面这段提示词可以直接发给 AI，让它继续协助你部署或排障：

```text
我在腾讯云服务器上部署一个 Node.js + Express + MySQL + Vite 前端项目，仓库是 riji。

项目信息：
1. 后端目录在 /var/www/riji
2. 前端目录在 /var/www/riji/app
3. 后端通过 npm start 启动，默认端口 3001
4. 前端构建命令是 cd /var/www/riji/app && npm run build
5. 数据库是 MySQL，库名 aa_diary
6. 我会用 Nginx 做反向代理，并给域名配置 HTTPS

你要按下面要求协助我：
1. 优先输出可以直接执行的命令，不要只讲概念
2. 如果需要改代码，明确告诉我要改哪个文件、哪几行、改成什么
3. 如果是排障，先帮我列出要检查的命令，再根据输出继续判断
4. 重点检查这几个风险点：
   - 前端 API_BASE 和 WS_URL 是否仍然指向 localhost
   - Nginx 代理 /api 和 WebSocket 是否配置正确
   - systemd 服务是否正常启动
   - MySQL 是否可连接，账号权限是否正确
   - 注册后配对进入主页是否仍有白屏
5. 如果你让我执行命令，请按“先执行什么，再执行什么”的顺序输出

当前我希望你帮我做的事情是：
[把你当前的具体问题写在这里，例如：帮我从零部署；或者：现在访问首页正常，但注册后进入配对页白屏，帮我排查]
```
